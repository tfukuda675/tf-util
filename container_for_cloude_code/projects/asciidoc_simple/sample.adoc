== はじめに

このドキュメントはAsciiDocとAsciidoctor PDFを使用してPDFを生成するサンプルです。

== 基本的なテキストフォーマット

=== 太字とイタリック

*太字のテキスト* と _イタリックのテキスト_ を使用できます。

=== リスト

順序なしリスト:

* 項目1
* 項目2
* 項目3
  ** ネストした項目
  ** もう一つのネストした項目

順序ありリスト:

1. 最初のステップ
2. 二番目のステップ
3. 三番目のステップ

== コードブロック

ソースコードを表示できます：

[source,python]
----
def hello_world():
    print("Hello, World!")
    return "success"

# 関数を実行
result = hello_world()
----

WARNING: これは注意が必要な内容です。

CAUTION: これは慎重に扱うべき内容です。

NOTE: これは補足説明です。

== PlantUML図表

=== クラス図

[plantuml]
....
@startuml
class ユーザー {
    - id: String
    - 名前: String
    - メール: String
    + ログイン(): Boolean
    + プロフィール更新(): void
}

class 商品 {
    - id: String
    - 商品名: String
    - 価格: Integer
    + 価格設定(): void
    + 在庫確認(): Integer
}

class 注文 {
    - id: String
    - 注文日: Date
    - 合計金額: Integer
    + 注文作成(): void
    + 支払い処理(): Boolean
}

ユーザー ||--o{ 注文 : 作成する
注文 }o--|| 商品 : 含む
@enduml
....

=== アクティビティ図

[plantuml]
....
@startuml
start
:注文受付;
if (在庫確認) then (在庫あり)
    :支払い処理;
    if (支払い完了?) then (成功)
        :配送手配;
        :完了通知送信;
    else (失敗)
        :エラー処理;
        :キャンセル通知;
    endif
else (在庫なし)
    :在庫切れ通知;
endif
stop
@enduml
....
